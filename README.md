# BOM

* Microcontroller (YD-RP 2040)
* Gateroon switches
* Hotswap module
* Gateroon stabilizer
* decoupling capacitors ( 100nF )
* 5V level shifter (74AHCT125)
* LEDs (SK 6812 mini E)
* keycaps
* Rotary encoder ( EC11E )
* Diodes ( 1N4148W )
* Oled screen ( 0.91 inch screen 128x32)
* Tantalum capacitor ( 470uF 6.3V D )
* Resistors ( Pack of many )
* Foam
* Coiled cable

# Assembly steps
* First the smaller parts must be souldered first because it will be harder to soulder the smaller components once the larger ones are placed / souldered.
* Secondly the larger components are souldered.
* Thirdly the foam is placed in the bottom case.
* After that all of the other parts are placed on the foam and closed off with the top plate.
* Finaly the cable can be connected to the PC and keyboard.
