# Anton-s-Keyboard

# BOM

* Microcontroller (YD-RP 2040)
* Gateroon switches
* Hotswap module
* Gateroon stabilizer
* decoupling capacitors ( 100nF )
* 5V level shifter (74AHCT125)
* LEDs (SK 6812 mini E)
* keycaps
* Rotary encoder ( EC11E )
* Diodes ( 1N4148W )
* Oled screen ( 0.91 inch screen 128x32)
* Tantalum capacitor ( 470uF 6.3V D )
* Resistors ( Pack of many )
* Foam
